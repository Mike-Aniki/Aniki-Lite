<h1 align="center">Aniki-Lite : Fullscreen Theme for Playnite</h1>

<div align="center">

**Aniki Lite** is a **fullscreen minimalist theme** for **Playnite**, **inspired** (and not a copy) by the **clean and functional design** of the **Nintendo Switch**.  

**Lightweight**, **smooth**, and **highly readable**, it is designed to run effortlessly on **any setup** â€” from **high-end gaming PCs** to **compact systems** and **handheld consoles**. It **automatically adapts** its layout for **16:9**, **16:10**, and **21:9** displays.  

Despite its **minimalist appearance**, **Aniki Lite** is **feature-rich** and supports popular extensions such as **NowPlaying**, **MoData**, **ThemeOptions**, **CheckDLC**, **SuccessStory**, and **SuccessStoryFullscreenHelper**.  

An **ideal choice** for those seeking a **fullscreen interface** that is **fast**, **clean**, and **minimalistic in style**.
